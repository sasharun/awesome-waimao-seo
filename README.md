## ⚡ 外贸站 SEO 最小可行清单

上线 2 周内先做完这些，再谈外链与内容矩阵：

1. **能被抓** — robots 不挡、sitemap 提交 GSC、HTTPS、可索引的 HTML 正文  
2. **说得清** — 首页 5 秒讲清卖什么/卖给谁；产品页有规格表而非空话  
3. **技术不拖后腿** — 移动端可用、LCP 别崩、无整站 JS 白屏  
4. **多语言不打架** — 每种语言独立 URL + hreflang 成对 + x-default  
5. **询盘接得住** — 表单可达邮箱、防垃圾、关键页都有 CTA（见 [转化清单](https://github.com/sasharun/dulizhan-inquiry-checklist)）

## 关键词调研工具

挖掘海外买家真正在 Google 搜的词,是外贸 SEO 的起点。下面这些工具覆盖搜索量、趋势、长尾与问题型关键词。

- [Google Keyword Planner](https://ads.google.com/home/tools/keyword-planner/) — 谷歌官方免费关键词规划师,需 Google Ads 账号,提供搜索量、竞争度与 CPC 估算。
- [Google Trends](https://trends.google.com/trends/) — 免费查看关键词随时间 / 地区 / 品类的热度变化,判断季节性与新兴趋势。
- [AnswerThePublic](https://answerthepublic.com/) — 把搜索词扩展成"谁 / 什么 / 为什么"等问题图谱,适合挖内容选题与长尾词。
- [AlsoAsked](https://alsoasked.com/) — 抓取 Google People Also Ask 问题与意图聚类,适合做外贸内容大纲和 FAQ 选题。
- [Keyword Surfer](https://surferseo.com/keyword-surfer-extension/) — 免费 Chrome 扩展,在 Google 搜索结果页实时显示搜索量与相关词。
- [Ubersuggest](https://neilpatel.com/ubersuggest/) — Neil Patel 出品的入门级工具,提供搜索量、关键词难度与内容创意,有免费额度可快速试用。
- [Keywords Everywhere](https://keywordseverywhere.com/) — 浏览器扩展,在搜索引擎与多个站点上叠加显示关键词数据,按量付费。
- [LowFruits](https://lowfruits.io/) — 批量分析 SERP 弱点与低竞争长尾词,适合新站寻找更容易排名的外贸内容机会。

## Google AI 搜索与收录加速（2025–2026）

- **Google AI Overviews** — 部分查询会出 AI 摘要；B2B 仍应以**可引用的规格/认证/案例内容** + 清晰实体信息取胜，别只堆关键词。
- [Google Search Central — AI features](https://developers.google.com/search/docs/appearance/ai-features) — 官方对 AI 结果与站长建议（以最新文档为准）。
- [IndexNow](https://www.indexnow.org/) — 内容更新后主动通知搜索引擎（Bing 等支持；可作补充，不能替代优质内容）。
- [Google Indexing API](https://developers.google.com/search/apis/indexing-api/v3/quickstart) — 主要用于 JobPosting / BroadcastEvent 等类型，**普通网页不要滥用**。
- 实操：新站优先 **GSC 网址检查 + sitemap**；改版用渐进式迁移与 301，而不是只指望 API。
- [FactoryBase](https://factorybase.ai) — 工厂目录站，GEO 落地比较完整的一个实例：每页答案胶囊 + 买家 FAQ + JSON-LD + `llms.txt`，并提供 MCP server 供 ChatGPT / Claude 直接查库。想参考"面向 AI 搜索的页面结构"可以扒它的源码。

## 多语言与 hreflang 国际化

外贸站常要做多语言,hreflang 配错会导致内容互相打架、排名错乱。

- [WPML](https://wpml.org/) — WordPress 最完整的多语言插件,自动生成 hreflang、多语言 URL 与分语种 sitemap,WooCommerce 多货币支持强。
- [Weglot](https://www.weglot.com/) — SaaS 自动翻译方案,支持 100+ 语言,自动处理 hreflang、翻译后 URL 与语言切换器,上线快。
- [Polylang](https://polylang.pro/) — 轻量级 WordPress 多语言插件,有免费版,自动 hreflang 与语言 URL,预算友好。
- [TranslatePress](https://translatepress.com/) — 前端可视化翻译,边看边改,自动生成 hreflang 与可索引的多语言页面。
- [Aleyda Solis Hreflang 生成器](https://www.aleydasolis.com/en/seo-resources-tools/hreflang-tags-generator/) — 国际 SEO 专家 Aleyda Solis 的免费 hreflang 标签生成工具。
- [TechnicalSEO Hreflang 工具](https://technicalseo.com/tools/hreflang/) — 生成并校验 hreflang 标签,避免漏掉自引用与 x-default。
- 🌟 多语言站的 hreflang 与子目录 / 子域结构是常见踩坑点,我用 Next.js 做多语言 SEO 站时会按市场规划目录与 hreflang,案例见https://modalcube.com。

## 询盘表单与防垃圾

询盘站的命门是表单——既要好填、又要挡住机器人垃圾询盘。

- [WPForms](https://wpforms.com/) — WordPress 拖拽式表单插件,内置 hCaptcha、honeypot、邮件通知,适合询盘场景。
- [Contact Form 7](https://wordpress.org/plugins/contact-form-7/) — 经典免费 WordPress 表单插件,可搭配 reCAPTCHA 与 honeypot 扩展。
- [Ninja Forms](https://ninjaforms.com/) — 灵活的 WordPress 表单构建器,支持条件逻辑与多种集成,适合复杂询盘流程。
- [Google reCAPTCHA](https://www.google.com/recaptcha/about/) — 谷歌验证码服务,v3 无感知评分可在不打扰真实客户的前提下过滤机器人。
- [hCaptcha](https://www.hcaptcha.com/) — 隐私友好的验证码与机器人防护方案,提供免费/商业版本和多种 CMS 集成。
- [Cloudflare Turnstile](https://www.cloudflare.com/products/turnstile/) — 免费、隐私友好的无感验证,reCAPTCHA 的轻量替代方案。

## ❓ 常见问题(FAQ)

### 外贸网站用什么程序做比较好?

主流是两条路:**WordPress** 上手快、插件生态成熟(WPForms、WPML、Rank Math 等),适合内容型与中小外贸站;**Next.js** 等现代框架性能与 Core Web Vitals 更好、谷歌SEO 上限高,适合追求速度与品牌质感的外贸独立站。预算和团队能力是关键变量,如需根据品类给具体方案,可邮件联系我。

### 外贸建站大概多少钱?

区间很大:模板套站 / 简单展示站从几千元起;带询盘表单、多语言、SEO 基础优化的定制站通常在一两万到数万元;复杂的多语言电商或工厂站更高。价格主要取决于页面数量、语言数、是否定制设计与后续 SEO 投入。建议先明确"要不要做谷歌 SEO 获客",再决定预算分配。

### 外贸 Google SEO 一般多久能看到效果?

通常 **3-6 个月**开始有稳定的自然流量与排名提升,竞争激烈的品类更久。外贸SEO 是复利型投入:前期做好技术基础、内容结构与关键词布局,中后期持续产出内容与外链,效果会越来越明显。想快速补量可同时跑 Google Ads。

### 没有流量怎么获取外贸询盘?

短期靠付费与平台:Google Ads(高意图搜索)、LinkedIn outreach、阿里巴巴国际站 / Made-in-China 等 B2B 平台;长期靠谷歌SEO + 内容 + 外链积累自然流量。最佳做法是"付费拉快流量 + SEO 养长期流量",并用询盘表单和数据追踪把转化跑通。

### 多语言外贸站 hreflang 必须做吗?

如果你针对多个语言 / 地区市场提供对应页面,**强烈建议做 hreflang**,否则 Google 可能在错误的语种结果里展示页面,甚至判为重复内容。WordPress 可用 WPML / Polylang / Weglot 自动生成,Next.js 可在路由层按市场配置。配置后务必用 hreflang 校验工具检查。

### 怎么防止外贸询盘表单收到大量垃圾邮件?

组合拳最有效:**reCAPTCHA v3 或 Cloudflare Turnstile(无感验证)+ honeypot 隐藏字段 + 邮箱实时校验 + 提交频率限制**。这样既能挡掉绝大多数机器人,又不打扰真实客户。

### 外贸 SEO 和 Google Ads 应该先做哪个?

不冲突,理想是并行:**Google Ads 见效快**,适合上线初期快速验证关键词与询盘转化;**外贸SEO 是长期资产**,需要时间但流量成本越来越低。预算有限时,可先用少量 Ads 测出高转化关键词,再围绕这些词做谷歌SEO 内容,把外贸推广与外贸获客的成本逐步压下来。
